# Mansi Portfolio

A premium recruiter-focused portfolio for a Data Scientist and Machine Learning Engineer.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide Icons
- Recharts

## Run Locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Personalization

- Replace `public/Mansi_Resume.pdf` with the final resume.
- Update project links, LinkedIn, GitHub, and email in `app/page.tsx`.
- Update experience, skills, education, achievements, and research content in `data/portfolio.ts`.
